# HanziMatch 复习系统设计方案

## 核心理念
基于 **可理解输入 + 间隔重复 (Spaced Repetition)**，让已学词汇在正确的时间点重复出现。

---

## 一、数据结构扩展

### 1. 词汇掌握度 (WordMastery)
```typescript
interface WordMastery {
  word: Word;           // 词汇详情
  learnedAt: number;    // 首次学习时间戳
  reviewCount: number;  // 复习次数
  correctCount: number; // 正确次数
  lastReviewed: number; // 上次复习时间
  nextReview: number;   // 下次复习时间（艾宾浩斯公式计算）
  level: 'new' | 'learning' | 'review' | 'mastered'; // 掌握等级
}
```

### 2. 掌握等级定义
| 等级 | 说明 | 条件 |
|------|------|------|
| new | 新词 | 从未学习 |
| learning | 学习中 | 正确 1-2 次 |
| review | 复习中 | 正确 3+ 次 |
| mastered | 已掌握 | 连续正确 5+ 次且超过 7 天未遗忘 |

---

## 二、复习算法 (SM-2 简化版)

### 复习间隔计算
```
首次复习: 1天后
第2次复习: 3天后  
第3次复习: 7天后
第4次复习: 14天后
第5次复习: 30天后 (达到 mastered)
```

### 错误处理
- 答错 → 重置间隔为 1 天，level 回退到 learning

```
│                 │
│  📊 学习统计     │ ← 新增
│     [已掌握 45 词]│
│                 │
└─────────────────┘
```

### 2. 复习模式
- 从 `learning` 和 `review` 状态的词中选取
- 优先选取 `nextReview <= now` 的词
- 每次复习 6-10 个词
- 答对更新下次复习时间，答错重置
